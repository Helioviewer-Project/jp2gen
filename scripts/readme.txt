jp2gen/scripts

Various helper scripts for the conversion of FITS data to JPEG2000 using JP2Gen.

- hv_aia_pull2.py

pull JP2 files from a remote website and make them ready for ingestion to the Helioviewer database.

- hv_aia_monitor.py

simple monitoring of the JP2 pull process.

- hv_jp2_delete_staged.py

script that deletes old files from the staging directory.
